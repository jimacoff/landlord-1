## Installation
Add this line to your application's Gemfile:

```ruby
gem 'landlord'
```

And then execute:
```bash
$ bundle install
$ bundle exec figaro install
```

Add your Stripe API keys to `config/application.yml`

Then execute:
```bash
$ rails landlord:import_plans
```
